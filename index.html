ABOUT US

The Shudder Shuttle is an immersive, one-of-a-kind escape room experience set in the bus itself. Each week, the bus changes to tell a new story that riders can play through. With voice actors, physical actors, smells and music, each room has been designed to envelop players in the story. Each winning team wins a set of free shirts in each member’s size. The Shudder Shuttle tours across Oklahoma, so be sure to visit and get out before we get away!


FOR NEWCOMERS

An escape room is an hour-long experience in which you and a team of friends are locked in a room and must escape. The thing that separates the Shudder Shuttle from all the rest is its fully immersive setting. The room is full of puzzles and riddles that will be used as your keys to escape in time. All codes and keys are only used once, so be sure to use your time wisely instead of trying to use the same code twice. Every key, clue or riddle is found through unlocking something, not smashing items. Once you’ve found the code to escape, it’ll be clear, so don’t worry about confusing it with another riddle.
